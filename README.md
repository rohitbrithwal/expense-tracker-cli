# Expense Tracker (Python)

A simple command-line based expense tracker built using Python.

## 🚀 Features

* Add expenses
* View all expenses
* Calculate total expense
* Menu-driven interface

## 🛠️ Technologies Used

* Python
* Lists
* Dictionaries

## ▶️ How to Run

1. Clone the repository
2. Run:
   python main.py

## 📌 Future Improvements

* Delete expense feature
* Search by category
* Save data to file (JSON)
